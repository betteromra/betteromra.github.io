Automation Programming for a Cozy Cardboard Crafting Game

Pawtumn Harvest is a 2.5D automation and crafting game developed in Unity during a game jam. Set in a cozy cardboard-inspired world, the game follows a group of black cats working together to automate the production of seasonal goods such as hot cocoa, pumpkin pie, and other autumn-themed recipes. The project blends hand-painted visuals, lightweight factory automation systems and accessible crafting mechanics into a stylized and charming gameplay experience.

As Lead Programmer and Gameplay Programmer, the project focused heavily on building the core automation system that drives the gameplay loop. This system enabled players to create production chains where resources are transported between crafting stations and transformed into finished goods. Special attention was placed on ensuring the system remained modular, intuitive and scalable, despite the extremely short development timeline of the game jam.

A key technical contribution was the development of the game's AI behavior system, which governs how worker cats interact with the world. The system supports dynamic task assignment, allowing AI agents to respond to production needs by collecting resources, operating crafting stations and maintaining efficient workflows. The goal was to create predictable yet flexible behavior that reinforced the player’s automation strategies.

In addition to gameplay and AI programming, contributions extended to UI programming, game design and art support, helping ensure consistency between mechanics and the game's handcrafted visual identity. The interface was designed to remain clear and readable while supporting rapid decision-making during automation planning and resource management.

Despite the two-week production window, Pawtumn Harvest successfully delivers a complete gameplay loop that combines automation, crafting and light strategy systems. The project highlights strong experience in gameplay systems design, AI programming and rapid prototyping in Unity, while emphasizing collaborative development in a fast-paced game jam environment.
